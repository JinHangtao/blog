About Me

Artist Statement · 关于此博客和我的想法

此博客网站使用 VS Code 开发 + GitHub 托管 + 後端即服務平台处理评论，应用程序分为移动端与桌面端，移动端原生版基于 Android Studio（Java）+ WebView 构建，Flutter 版基于 Dart 跨平台框架；桌面端 Windows App 基于 Electron（Node.js）+ WebView 构建，Tauri App 基于 Rust 框架，均通过 GitHub Releases 分发，于19嵗开发（2025-2026），為記錄blog日常以及保存分享自己在18嵗前的創作和回憶，今天，我們還年輕。

博客的作品页里的内容，创作于13至18岁之间——在鲁迅美术学院附属中学的画室里，使用铅笔、炭笔、油画，色粉等工具，隨后在製作作品集時拓展至三维建模。後來18嵗考入罗马国立美术学院，接觸到聲音設計，摄影与 AI 生图等媒介，還在探索。

如果绘画是被動的和服從的，我能夠明顯感覺到是一條我会后悔和難過的路，所以在美院報考時选择了其他的專業，比如藝術新技術。

這裏面的作品都是18嵗包括18嵗以内創作的，繼續探索一些更有意思的東西和工具留在blog這裏或者單獨創建平臺，謝謝屏幕前的各位嘻😄。

另一个版本的博客,个人很喜欢且制作挺久,不过有些花哨不适合长期,在这https://jinhangtao.github.io/reliquary。

· Android App 是原生版本，Flutter App 是跨平台框架版当前仅 Android；桌面端推荐 Windows App，Tauri App 体积更小但外部链接暂时打不开(下载时若遇安全提示，点击右侧 ··· "保留"可正常安装)

40多亿年的故事

这个音频作品3分10秒压缩了40多亿年的地球初期历史，用声音讲地球从一开始到多细胞生命出现的故事。 前面是地球刚形成时的混乱和喷发的岩浆，用尖锐的声音表现；然后慢慢听到水流声和气泡声， 像单细胞生命慢慢冒出来；最后多细胞生命登场，从海洋生物、鸟类、爬行动物，到人类，低沉的声音让这一切听起来很史诗，很有感觉。

2025-2026 Interactive Video

Dreaming Inside an Elastic Space - 彈性空間

《Dreaming Inside an Elastic Space》是一個互動式影片作品，透過「彈性空間」的概念，探索憂鬱與自殺意念——也就是人的內在心理狀態如何實際改變周圍的世界。 作品沒有戲劇化或誇張的表現，而是讓觀者親身進入這些情緒狀態。你的每個選擇都會影響空間：物體變得更重或更輕，房間收縮、擴張、扭曲，聲音也模糊了思想與環境的界線。 每一次決定都會重新塑造「心靈與空間」之間的彈性關係，而你最後抵達的地方，反映了你在這個可變世界中的選擇與路徑。 Dreaming Inside an Elastic Space is interactive video explores elastic space through the lens of depression and suicidal ideation — how interior psychological states physically reshape the world around us. There is no spectacle, no dramatization. Instead, you inhabit these emotional conditions, making choices that demonstrate how space itself becomes elastic under psychological pressure. Objects gain or lose weight. Rooms contract, expand, distort. Sound bends the boundaries between thought and environment. Each decision reconfigures the elastic relationship between mind and space. Where you arrive reflects how you've navigated this malleable reality. Created by Jinhantao Digital Video, Accademia di Belle Arti di Roma.
